URL Shortener API

Documentation and Usage Guide

Base URL (Local): http://localhost:3000

Base URL (Hosted): https://url-shortener-3n6e.onrender.com

Google Authentication

  • GET /auth/google

    Initiates Google Sign-In for user authentication.

  • GET /auth/google/callback

    Callback URL for Google Sign-In.

URL Management

  • POST /api/shorten

    Creates a shortened URL from a long URL.

  • GET /api/shorten/:alias

    Redirects to the original long URL for the given alias.

Analytics

  • GET /api/analytics/:alias

    Retrieves detailed analytics for a specific short URL.

  • GET /api/analytics/topic/:topic

    Retrieves analytics for all short URLs under a specific topic.

  • GET /api/analytics/overall

    Retrieves overall analytics for all URLs created by the user.

Rate Limiting

- A maximum of 10 requests per 10 seconds is allowed per IP address.
- Exceeding the limit will block the IP for 10 seconds.
